# Route hero photos

One image per route, named `<route-id>.jpg`, referenced by `photo_url` in
`../data/routes.geojson`. Film/documentary frames — not stock.

Expected files for the current routes:

- `beerwah-state-forest-loop.jpg`
- `marburg-glamorgan-vale-gravel.jpg`
- `mount-mee-forest-run.jpg`

The detail panel hides the image gracefully if a file is missing, so the map
works before photos are dropped in. Recommended: ~1200×800, optimized JPG.
